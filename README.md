# DallNet Analytical Chemistry Lab Pro

מערכת מתקדמת לניהול מעבדה כימית, ביצוע חישובים אנליטיים והפקת דוחות.
המערכת פותחה כחלק מ-**Ali Dallasheh Smart Learning Systems 2026**.

## תכונות מרכזיות
*   **ממשק מאובטח:** ניהול תנאי שימוש וכניסה מאובטחת.
*   **כלים אנליטיים:** חישובי LINEST, המרת יחידות וניתוח נתונים.
*   **ניהול דוחות:** תצוגה מקדימה והדפסה מותאמת של דוחות מעבדה.
*   **עיצוב רספונסיבי:** התאמה מלאה למחשבים ניידים וטאבלטים.

## טכנולוגיות
*   **Frontend:** HTML5, CSS3 (עם משתני CSS ו-Flexbox/Grid), JavaScript.
*   **פיתוח:** מבוסס על סטנדרטים של `Ali Dallasheh Smart Learning Systems`.

## התקנה
1. הורד את הקבצים לפרויקט מקומי.
2. פתח את `index.html` בדפדפן מודרני.
3. אשר את תנאי השימוש כדי לגשת למערכת.

## זכויות יוצרים
כל הזכויות שמורות ל-Ali Dallasheh. השימוש במערכת מותנה בהסכמה לתנאי השימוש המפורטים בממשק.
